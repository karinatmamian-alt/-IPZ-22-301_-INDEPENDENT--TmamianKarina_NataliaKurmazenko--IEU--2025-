Самостійна робота №1

1. Тема

HTML-документ. Семантична розмітка. GitHub. GIT. Робота з репозиторіями. Опис бізнес-логіки власного WEB-застосунку.

2. Мета

Набути практичних навичок роботи з GitHub, репозиторіями, HTML-документами, таблицями, зображеннями, посиланнями, списками та формами. Створити шаблон звітного HTML-документа для відображення результатів усіх лабораторних робіт.

3. Хід виконання роботи

  • Вибрати предметну галузь майбутнього WEB-застосунку та описати її функціонал.
  • Описати бізнес-логіку у вигляді короткого сценарію.
  • Зареєструватися у програмі Figma та пройти 9 коротких уроків.
  • Створити макет WEB-застосунку у Figma.
  • Встановити Git, зареєструватися на GitHub, встановити GitHub Desktop.
  • Створити репозиторій власного WEB-застосунку.
  • Створити посилання на живу сторінку (GitHub Pages).
  • Склонувати проєкт із GitHub на свій комп’ютер.
  • Відкрити проєкт у Visual Studio Code, створити файл index.html.
  • Написати HTML-код головної сторінки за макетом із Figma, використовуючи заголовки, таблиці, зображення, списки, форму та посилання.
  • Створити додаткову сторінку з описом предметної галузі та бізнес-логіки.
  • Запушити проєкт на GitHub і перевірити роботу через посилання на живу сторінку.

4. Висновки

Під час виконання самостійної роботи №1 ми ознайомились з основами HTML, створили власний файл index.html, навчились користуватися GitHub, GitHub Desktop і Visual Studio Code, а також створювати структуру HTML-документа для подальших лабораторних робіт.
